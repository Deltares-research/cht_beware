# Coastal Hazards Toolkit - BEWARE

Welcome to the GitHub page of the Deltares CHT BEWARE.

To use the beware package, you could create an environment with:

mamba create -n beware python=3.11

and then install the BEWARE package with:

cd d:\checkouts\github\cht_beware
pip install .

For an editable package, install the package with e.g.:

cd d:\checkouts\github\cht_beware
pip install -e .

or something like:

pip install -e d:\checkouts\github\cht_beware
